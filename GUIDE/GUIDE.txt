First, when you turn on the program, you will need to go to the terminal and type the command "npm install" to be able to use it. Then type "npm start" to run the application, depending on the device you want to use, "a" to turn on the virtual machine, "w" to use on the web.
Once you can access the application's home page, here you can add new data by clicking the "+" sign in the top right corner, and fill in the data you want. It should appear in the All Expenses section, and in Recent (if it happened in the last 7 days). You can also correct the data if there are errors during the checking process. By clicking on the incorrect data, you will be taken to the editing screen and can add, remove, edit or delete them.
